# 呼朋引伴環島趣工具

這是一個不碰 Apple/Google 上架流程的純前端工具。直接用瀏覽器開啟 `index.html` 即可使用。

## 工具範圍

- 免登入，不收 Email、不收授權碼。
- 不做裝置綁定，只產生匿名 ID，存在瀏覽器 localStorage。
- 移動方式：機車、自行車、重機、徒步。
- 使用者手動更新定位，不做背景定位。
- 可用 Google Maps 開啟自己的目前位置，不提供附近車友精準座標。
- 位置防盜可設定停放點與警戒半徑，手動檢查目前位置是否超出範圍；需先更新定位。
- 附近雷達以距離圈層顯示，不代表真實方向：一般 1/3/10km，徒步 1/5/15km，機車與重機 3/10/30km，需要協助 5/15/50km。
- 公開工具使用人次會顯示給所有使用者；本機/區網測試不累計，正式部署後才累計首頁開啟次數，存在 `usage.json`，不記錄 IP、定位、裝置指紋或個人資料。
- 揪團卡固定 120 分鐘自動過期。
- 附近車友顯示約略距離，不顯示精準座標。
- 模板訊息加短文字補充，用於快速傳達需求。
- 行前檢核依移動方式切換，可勾選並確認檢核完成；徒步包含徒步環島準備項目。
- 可一鍵刪除本機資料。

## 工具限制

目前沒有後端，所以附近車友與檢舉停權是前端展示資料。若要變成可上線的多人版本，需要補：

- 後端 API 與資料庫。
- 位置資料 120 分鐘 TTL 或定時清除。
- 檢舉、封鎖、停權資料流。
- 隱私政策與服務條款頁。
- HTTPS 部署，否則部分瀏覽器不允許定位。

## 安全邊界

「需要協助」不是緊急救援服務。正式版應在需要協助卡片附近放 110/119 提示，並避免任何救援承諾。

## 降低法律風險的文字與流程

- 定位前需勾選同意：只在本次使用附近功能時取得定位，不做背景定位，不公開精準座標。
- 畫面明確提示：請勿填寫真名、電話、Email、車牌或其他可直接識別個人的資料。
- 揪團卡旁提示：卡片 120 分鐘自動過期，「需要協助」不等於報案或救援派遣。
- 安全區塊提示：本服務不提供醫療、道路救援、交通管制或安全保證，危急請撥打 110/119。
- 使用者可隨時關閉位置或刪除本機資料。
- `legal-notice.html` 放工具版服務與隱私告知；正式上線前需依實際資料流請律師修訂。

# 呼朋引伴繞境環島趣工具

這是一個不碰 Apple/Google 上架流程的純前端工具。直接用瀏覽器開啟 `index.html` 即可使用。

## 工具範圍

- 免登入，不收 Email、不收授權碼。
- 不做裝置綁定，只產生匿名 ID，存在瀏覽器 localStorage。
- 移動方式：機車、自行車、重機、徒步。
- 使用者手動更新定位，不做背景定位。
- 可用 Google Maps 開啟自己的目前位置，不提供附近使用者精準座標。
- 位置防盜可設定停放點與警戒半徑，手動檢查目前位置是否超出範圍；需先更新定位。
- 附近雷達顯示距離圈層、自己的定位狀態，以及附近使用者的約略方位點，不顯示他人精準座標。
- 公開工具使用人次會顯示給所有使用者；本機/區網測試不累計，正式部署後透過 Cloudflare Pages Functions + KV 累計首頁開啟次數，不記錄 IP 或裝置指紋。
- 揪團卡固定 120 分鐘自動過期。
- 附近使用者透過 Cloudflare Pages Functions + KV 顯示匿名列表，只回傳暱稱、車種、狀態、約略距離、量化方位與最後更新時間，不回傳精準座標。
- 模板訊息加短文字補充，用於快速傳達需求。
- 行前檢核依移動方式切換，可勾選並確認檢核完成；徒步包含徒步環島準備項目。
- 可一鍵刪除本機資料。

## 工具限制

目前多人附近功能使用 Cloudflare KV 作為 MVP 資料庫，但仍有正式版限制：

- 位置資料以約略座標保存，30 分鐘未更新即不顯示。
- 無登入與身分驗證，暱稱仍可能被冒用。
- 檢舉、封鎖、停權資料流。
- 隱私政策與服務條款需依實際營運資料流修訂。
- 需 HTTPS 部署，否則部分瀏覽器不允許定位。

## Cloudflare KV 設定

正式部署後若要啟用公開使用人次與附近使用者列表：

- 在 Cloudflare 建立 KV namespace，例如 `hu_peng_yin_ban_usage`。
- 到 Pages 專案的 Settings -> Functions -> KV namespace bindings。
- 新增 binding，Variable name 必須填 `USAGE_KV`。
- 綁定剛建立的 KV namespace。
- 重新部署 Pages。

同一個 KV binding 目前保存：

- `public-visits`：公開工具使用人次。
- `nearby-users`：30 分鐘內更新的匿名附近資料。

## 降低法律風險的文字與流程

- 定位前需勾選同意：只在本次使用附近功能時取得定位，不做背景定位，不公開精準座標。
- 附近功能僅回傳約略距離與量化方位，不回傳他人座標；後端僅保存約略座標並短時間顯示。
- 畫面明確提示：請勿填寫真名、電話、Email、車牌或其他可直接識別個人的資料。
- 揪團卡旁提示：卡片 120 分鐘自動過期，「需要協助」不等於報案或救援派遣。
- 安全區塊提示：本服務不提供醫療、道路救援、交通管制或安全保證，危急請撥打 110/119。
- 使用者可隨時關閉位置或刪除本機資料。
- `legal-notice.html` 放工具版服務與隱私告知；正式上線前需依實際資料流請律師修訂。
